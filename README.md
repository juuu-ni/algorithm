# 📘 Algorithm Study Repository

이 저장소는 주요 알고리즘을 Python으로 구현하고 정리한 학습용 레포지토리입니다.  
기본 개념 이해 및 문제 풀이 연습을 목적으로 작성되었습니다.

---

## 📂 파일 구성

| 파일명 | 설명 |
|--------|------|
| BFS.py | 너비 우선 탐색 (Breadth-First Search) |
| DFS.py | 깊이 우선 탐색 (Depth-First Search) |
| DP.py | 동적 계획법 (Dynamic Programming) |
| MST.py | 최소 신장 트리 (Minimum Spanning Tree) |
| 그리디.py | 그리디 알고리즘 (Greedy Algorithm) |
| 다익스트라.py | 다익스트라 최단 경로 알고리즘 |
| 플로이드.py | 플로이드-워셜 알고리즘 |
| 이진탐색.py | 이진 탐색 (Binary Search) |
| 투포인터.py | 투 포인터 알고리즘 |
| 백트래킹.py | 백트래킹 알고리즘 |
| 시뮬레이션.py | 시뮬레이션 문제 풀이 |
